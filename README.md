# Minutely

**Meeting Intelligence for Modern Teams**

Minutely is a meeting management app built to help teams schedule meetings, capture notes, extract action items, and keep meeting history organized in one place.

## Goals

- Make meetings easier to manage
- Turn conversations into clear notes and action items
- Keep meeting records searchable and organized
- Help teams track decisions and follow-ups
- Create a clean, productivity-focused workspace

## Core Features

- Dashboard for upcoming and recent meetings
- Meeting calendar
- Team chat and collaboration
- AI-generated meeting notes
- Speech-to-text transcription
- Action item and decision extraction
- Meeting archive
- Searchable meeting history

## Tech Direction

- **Frontend:** React / Electron
- **Backend API:** Go
- **AI Services:** Python
- **Database:** Supabase

## Vision

Minutely aims to turn meetings into structured, actionable knowledge for modern teams.
